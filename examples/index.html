OSM MCP Server Tools

1. Start the MCP Inspector

The MCP Inspector is needed to connect to the OSM MCP Server. Run this command in your terminal:

npx @modelcontextprotocol/inspector uv run --with mcp --with haversine --with aiohttp --with shapely --with geojson mcp run src/osm_mcp_server/server.py

After running the above command:

  1. The MCP Inspector proxy server will start on port 3000
  2. The MCP Inspector web client will be available at port 5173
Open MCP Inspector Web Client

How It Works

This setup uses two key components:

  1. MCP Inspector: Connects to the OSM MCP Server and provides a way to call its tools.
  2. OSM Client: Our visualization tool that uses the MCP Server's capabilities to display maps, directions, and other location-based information.

You can use both tools together to explore and visualize OpenStreetMap data.
